# Smart Waste Management System

## Project

IoT Smart Waste Management System using Raspberry Pi Pico.

## Objective

To monitor garbage level in real time using an ultrasonic sensor and upload data to the cloud.

## Hardware

- Raspberry Pi Pico W
- HC-SR04
- Breadboard
- Jumper Wires

## Software

- Thonny IDE
- MicroPython
- ThingSpeak

## Features

- Real-time monitoring
- IoT Dashboard
- Alert when bin is full
